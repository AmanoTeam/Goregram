# Goregram

Goregram is a fork of Telegram, a messaging app focused on speed, security, and customization.

## Features

- Allow saving and copying messages from chats with **Restrict saving content** enabled
- Unlock local Telegram Premium features for free
  - Enable custom app icons
  - Remove sponsored messages
  - Increase download speeds
- Allow logging into up to 16 accounts simultaneously
- Always open files in a separate task

## Privacy Features

- Spectator Mode
  - Hide your typing, online, and read status from other users until you interact with them
